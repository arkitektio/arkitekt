actors.utils

log​

def log(message: str, level: str = "info")

Logs a message

Depending on both the configuration of Arkitekt and the overwrite set on the Assignment, this logging will be sent (and persisted) on the Arkitekt server

Arguments:

  • message sr - The Message you want to send
  • level DebugLevel, optional - The level of the log. Defaults to DebugLevel.INFO.

Returns:

  • [Future] - Returns a future if currently running in an event loop
